# Tutorial

## Set up stuff

### Proper OS

    virtualenv venv
    source venv/bin/activate
    export FLASK_APP=hello_flask.py
    export FLASK_ENV=development

### Windows

    virtualenv venv
    source venv/Scripts/activate
    export FLASK_APP=hello_flask.py
    export FLASK_ENV=development

## run server

    flask run
